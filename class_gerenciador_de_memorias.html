Descrição detalhada

Classe responsável por criar e salvar listas informadas pelo usuário.

Essa classe é usada para registrar as listas informadas pelo usuário e aquelas que são criadas durante operações que o usuário realiza. Essas listas podem ser recuperadas para serem novamente usadas em operações pelo usuário. Essa classe também realiza o salvamento em arquivo das listas que estão na memória.
