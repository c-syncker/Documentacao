Detailed Description

Classe responsável por tratar as entradas do usuário.

Essa classe recebe listas numéricas e/ou alfanuméricas do usuário. Faz o tratamento necessário sobre essa entrada extraindo listas de alturas e durações a partir dos valores informados. Essas listas podem ser passadas para outras classes que realizam operações sobre listas.
